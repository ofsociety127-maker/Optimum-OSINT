# Optimum-OSINT

Optimum-OSINT is a multi-purpose open-source intelligence toolkit built for gathering and analyzing public information across domains, websites, IPs, DNS records, phone numbers, and social platforms such as Discord. It provides automated reconnaissance, data collection, parsing, and investigation utilities in a single framework.

## Features

- Website and metadata scraping
- Domain and WHOIS lookups
-   Discord, telegram
- DNS resolution and reverse DNS checks
- IP and hostname investigation
- Phone number parsing and carrier/geolocation support
- Discord-focused OSINT workflows
- CSV, SQLite, and JSON data handling
- Multithreaded scanning and task execution

## Tech Stack

- Python 3
- Requests
- BeautifulSoup4
- SQLite3
- ThreadPoolExecutor
- Whois
- dnspython
- phonenumbers
